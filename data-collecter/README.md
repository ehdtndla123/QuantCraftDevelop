# 코인 관련 데이터를 수집하는 서버입니다.

# Requirements
- Python 3.9
- Ubuntu 20.04
- Kafka

# Installation
1. Install Python 3.9
2. Install Kafka
3. Install requirements
```bash
pip install -r requirements.txt
```

# Start
```bash
python main.py
```

# Description
- 코인 관련 데이터를 수집하는 서버입니다.
- 코인 관련 데이터를 수집하여 Kafka에 전송합니다.
